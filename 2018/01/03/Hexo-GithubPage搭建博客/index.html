Hexo+GithubPage搭建博客

1.安装git（ssh连接github）
2.安装Node.js
3.安装Hexo（建立文件夹Blog，npm install -g hexo-cli安装hexo，hexo init Myblog，在Myblog命令行下hexo s得到本机初始网页）
4.关联github（建立zhangdoubledong.github.io仓库，再修改Myblog下的_config.yml中deploy，再git config – global user.name “zhangdoubledong”和邮箱，
再hexo s，npm install hexo-deployer-git –save，hexo clean，hexo g，hexo d则可以用网络浏览器浏览zhangdoubledong.github.io）
5.绑定域名（阿里云一元购域名，然后解析，CNAME中使用ping zhangdoubledong.github.io得到github的IP，修改仓库的Custom domain，在Myblog文件下的source中
新建全格式的CNAME文件夹，内容为zhangdoubledong.tech，然后hexo clean，hexokinase个，hexo d就可用个性化域名浏览网站）
6.创建文件（hexo n “时间”，hexo s，hexo g，hexo d）
